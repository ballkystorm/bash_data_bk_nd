# BashData

BashData is a modern backend platform for data subscription and digital services built with Java and Spring Boot.

## Current Status

Current Version: v0.1.0

Completed Features

- User Registration
- Spring Security Configuration
- Global Exception Handling
- Business Error Framework
- Business Success Framework
- Standard API Response
- Request Validation
- Password Encryption (BCrypt)
- Email Normalization
- Phone Number Normalization
- User Code Generation
- MapStruct Mapping
- PostgreSQL Integration
- JPA Auditing
- Centralized Logging

## Technology Stack

- Java 21
- Spring Boot 3
- Spring Security
- Spring Data JPA
- PostgreSQL
- MapStruct
- Lombok
- Maven
